1. Breif intro to HTML

HTML ( HyperText Markup Language, a.k.a “How To Meet Ladies” a geek joke from Silicion Valley ) is a standard markup language used to create web pages. Compare to coding language, HTML is a set of markup tags used to describe a webpage. A typical HTML file usually ends up with .html or .htm which can be viewed by a browser. For example:

  • <!DOCTYPE html>: claims as HTML5 document, “html” is case-incensitive.
  • <html>: root, defines the whole HTML document.
  • <head>: contains meta data as <meta charset="utf-8"> and define the coding format as utf-8, sometimes you run into a messy code, check the coding format. For more info, please refer to ref2
  • <title>: contains the title of the document
  • <body>: defines the main content
  • <h1>: defines a heading
  • <p>: defines a paragraph

2. Tags in HTML

The tags is formatted as <keyword>, eg <html>, and should be paired, eg <b> and </b>, which is called the opening tag and close tag respectively. Though Tags is case-insensitive, but lower case is usually preferred.

2.1 Heading and paragraph

2.1.1 Heading

use <h1> - <h6>, use heading only if you want to set a heading, not for the bold or size purpose. eg:

<h1>This is a heading</h1>

2.1.2 Paragraph

use <p> to start a new paragraph, if you want to start a new line without starting a new paragraph, you can use <br>, eg:

<p>This is a paragraph</p> \
<p>This is <br> a <br> paragraph</p>

2.1.3 Others

use <hr> to set horizontal line in contents.

use <!-- --> to add comments, which is ignored by the browswer and not shown

<!-- This is a comment -->

when displaying, browser will automatically remove extra space and lines in HTML files

empty elements contains no content, it was closed by / in the opeing tag, like the image above, also <br /> as a break.

2.3 Links

use <a>, href is link address , set target="_blank" to open the link in a new Tab, eg:

<a href="http://www.vinkwok.github.io" target="_blank">This is my blog link</a>

use id to create a bookmark within the HTML page, eg:

<a id="tips">Tips</a> \
<a href="#tips">Go to tips in the same page</a> \
<a href="http://www.runoob.com/html/html-links.html#tips"> Go to tips from the other page</a>

2.4 Tables

use <table> to define a table, use <tr> to define a row, <td> to define a cell (‘td’ means ‘table data’), <th> to define the header. Example:

2.5 Images

<img> is an empty tag, name and size is defined by the attribute. eg:

<img src="/images/logo.png" alt="logo" width="258" height="39" />
use src to show the pic, use alt to replace the image with text when image cannot be displayed, use height and width to denote the length and width of the image (usually a good habit to do so)
